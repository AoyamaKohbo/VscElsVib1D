一自由度粘弾性振動系

一自由度粘弾性振動系の振動(運動)方程式は次式となります。

\begin{align} m\ddot x + c\dot x + kx = f,~~~y=x. \label{eq:VivEq} \end{align}

ここに、\(m\)は質量、\(c\)は粘性係数、\(k\)は弾性係数です。

\(a=c/m\)、\(b=k/m\)、\(bu=f/m\)とおくと次式が得られます。

\begin{align} \ddot x + a\dot x + bx = bu,~~~y=x \label{eq:VivEq1} \end{align}

固有角振動数\(\omega_n=\sqrt{k/m}\)なので、\(b=\omega_n^2\)です。 また、\(a=2\zeta\omega_n\)と表記します。

式\eqref{eq:VivEq1}を状態方程式 で記述すると次式になります。

\begin{align} \left[ \begin{array}{@{\,}l} \dot x\\ \ddot x \end{array} \right] = \left[ \begin{array}{@{\,}cc} 0&1\\ -b&a \end{array} \right]\left[ \begin{array}{@{\,}l} x\\ \dot x \end{array} \right] + \left[ \begin{array}{@{\,}l} 0\\ b \end{array} \right]u,~~~ y=[1~~0] \left[ \begin{array}{@{\,}l} x\\ \dot x \end{array} \right]. \end{align}

そこで、状態ベクトル\(\mathbf{x}\)と係数行列\(\mathbf{A}\)、\(\mathbf{B}\)、\(\mathbf{C}\)を次式のように設定します。

\begin{align} \mathbf{x} =\left[ \begin{array}{@{\,}l} x\\ \dot x \end{array} \right],~~~ \mathbf{A} = \left[ \begin{array}{@{\,}cc} 0&1\\ -b&-a \end{array} \right],~~~ \mathbf{B} = \left[ \begin{array}{@{\,}l} 0\\ b \end{array} \right],~~~ \mathbf{C}=[1~~0]. \end{align}

状態ベクトル\(\mathbf{x}\)と係数行列\(\mathbf{A}\)、\(\mathbf{B}\)、\(\mathbf{C}\) を用いて一自由度粘弾性振動系\eqref{eq:VivEq}を次式のように連続時間状態方程式で記述きます。

\begin{align} \left\{ \begin{array}{@{\,}cl} \dot{\mathbf{x}} ~=& \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{u}\\ \mathbf{y} ~=& \mathbf{C}\mathbf{x} \end{array} \right. \label{eq:StateEq} \end{align}

サンプリング周期\(T=0.005\)[s]、\(\omega_n=6.28×10\)[rad/s]、\(\zeta=0.2\)とすると、 \(a=25.12\)、\(b=3943\)です。

\begin{align} \mathbf{A} = \left[ \begin{array}{@{\,}cc} 0&1\\ -3943&-25.12 \end{array} \right],~~~ \mathbf{B} = \left[ \begin{array}{@{\,}c} 0\\ 3943 \end{array} \right]. \end{align}

連続時間空間表現\eqref{eq:StateEq}の係数行列\(\mathbf{A}\)、\(\mathbf{B}\)を 離散時間状態方程式 の係数行列\(\mathbf{P}\)、\(\mathbf{Q}\)に変換します。

\begin{align} \mathbf{P}=e^{\mathbf{A}T}= \left[ \begin{array}{@{\,}cc} 0.9531&0.004222\\ -18.23&0.8370 \end{array} \right],~~~ \mathbf{Q}=\mathbf{A}^{-1}(e^{\mathbf{A}T}-\mathbf{I})\mathbf{B}= \left[ \begin{array}{@{\,}c} -0.04691\\ 18.23 \end{array} \right]. \label{eq:RtnEqn} \end{align}

式の\eqref{eq:RtnEqn}係数行列\(\mathbf{P}\)、\(\mathbf{Q}\)を用いて 一自由度粘弾性振動系\eqref{eq:VivEq}は次式の離散時間状態方程式で記述されます。

\begin{align} \left[ \begin{array}{@{\,}l} x(n+1)\\ \dot x(n+1) \end{array} \right] = \left[ \begin{array}{@{\,}cc} 0.9531&0.004222\\ -18.23&0.8370 \end{array} \right]\left[ \begin{array}{@{\,}l} x(n)\\ \dot x(n) \end{array} \right] + \left[ \begin{array}{@{\,}c} -0.04691\\ 18.23 \end{array} \right]u(n),~~~ y(n)=[1~~0] \left[ \begin{array}{@{\,}l} x(n)\\ \dot x(n) \end{array} \right]. \end{align}
